# Coding Portfolio Website

A personal portfolio website built with HTML, CSS, and JavaScript. Features a modern dark theme design with responsive layout.

## 📁 Files

- `index.html` - Homepage
- `about.html` - About page
- `projects.html` - Projects page
- `contact.html` - Contact page
- `styles.css` - Complete stylesheet

## 🚀 Getting Started

1. Open `index.html` in a web browser to view the site
2. Customize content as needed (see customization guide)
3. Deploy to GitHub Pages for public hosting

## 🎨 Customization

See `README.md` in the Website folder for detailed customization instructions.
